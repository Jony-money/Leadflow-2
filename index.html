One-time setup — about 5 minutes
Step 1 of 3 — Anthropic API Key
Get your key from console.anthropic.com/keys. Stored only in your browser — never leaves your machine.
Step 2 of 3 — Google OAuth Client ID
1. Google Cloud → Credentials
2. Create OAuth 2.0 Client ID → Web application
3. Authorized origin: http://localhost:8765
4. Enable Google Calendar API and Gmail API
Step 3 of 3 — Connect Google Account
Sign in with Google to grant access to Calendar and Gmail.
